# Расширение для браузера Chrome

Расширение которое добавляет красивую панель справа в голосовый чат рулетке Nekto.me

Функционал:
- Автоскип собеседника
- Смена темы сайта
- Статистика
  - Разговоров всего
  - Разговоров за сегодня
  - Самый длинный разговор

# Установка

1. Скачиваете весь проект
2. Из архива переносите все файлы в отдельную папку
3. Заходите по ссылке - chrome://extensions/
4. В этой ссылке будет кнопка (Режим разработчика) ее нужно включить
5. Нажимаете на (Загрузить распоковонное расширение)
6. Выбираете папку с расширением
